Game

About

The idea of the game comes from web games with the same name "Flappy 2048", which is a mix of famous games "Flappy Bird" and "2048" in 2014.

Description

Player has to control a "bird" which is a box containing a number. The "bird" is moving towards some pillars which is formed by a number of boxes. If the "bird" hits on the boxes which having a different number with the "bird" or the hidden part of pillar above the 6 boxes, the game is over. Only the boxes having the same number as the "bird" can permit player passing through the pillar. Each time the "bird" passes through the pillar successfully, the score will increase by 1.
The "bird" will fall down if player does not do anything. To keep the "bird" flying, users have to keep pressing the jump button to maintain the flying height of the "bird.

Control

"Jump" is the only control in the game. Player can make the "bird" jumps by pressing any buttons on keyboard or mouse.

Compile & Run

To compile the source code of Flappy 2048, please run this command under 'src' folder with JDK version 8 or above:
javac model/*.java view/*.java controller/*.java flappy2048injavafx/*.java
After compiling the source code, the game can be start by running this command under 'src' folder:
java flappy2048injavafx.Flappy2048InJavaFx

Preview


Player can press on the start button or space bar to start the game.

The "bird" controlled by the player is the box with "2" on the left hand side. A pillar is formed by 6 boxes. The "bird" has to pass the pillar through the box with the same number (i.e. "2") in the pillar.

If the "bird" successfully passed the pillar, the number inside the "bird" will be doubled. After that, the player can keep continue to pass other pillars through the box with the same number.

If the "bird" hit the boxes without the same number as the "bird", the game is over.

The game over screen will show your final score and the top 3 scores you have got. You can choose "Restart" to play the game again or "Quit" to exit the game.
